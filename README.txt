SUKI RAE GMAIL SIGNATURE PACKAGE

1. Create a free GitHub account.
2. Create a public repository named: suki-signature
3. Upload these files:
   - index.html
   - suki-rae-email-signature.png
   - gmail-signature-template.html
4. Go to Settings → Pages.
5. Source: Deploy from branch.
6. Branch: main / root.
7. Wait 1–2 minutes.
8. Your image URL will be:
   https://YOUR-GITHUB-USERNAME.github.io/suki-signature/suki-rae-email-signature.png
9. Open gmail-signature-template.html.
10. Replace YOUR-GITHUB-USERNAME with your real GitHub username.
11. Open the edited HTML file in Safari/Chrome.
12. Copy the signature area.
13. Paste into Gmail Settings → Signature.
14. Save changes.

Important: the banner is exactly the design image. For Gmail reliability, the image must be hosted online.
